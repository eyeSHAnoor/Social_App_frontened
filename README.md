# Social_App_fMERN Stack Social Media App
Overview
This is a full-stack social media application built using the MERN stack (MongoDB, Express.js, React, and Node.js). The app includes functionalities for user authentication, posting content, viewing posts, managing followers, and more. It uses React Router for navigation and Bootstrap for styling.

Features
User Authentication: Users can sign up, log in, and access protected routes.
Create and Manage Posts: Users can create new posts and view them in a feed.
User Profiles: Users have personal profiles where they can view their posts.
Follow and Unfollow: Users can follow and unfollow other users.
View Followers and Following: Users can see their followers and whom they are following.
Error Handling: Custom error pages for not found routes.
Technologies Used
Frontend: React, React Router, Bootstrap
Backend: Express.js
Database: MongoDB
Authentication: JWT (JSON Web Tokens)rontened
